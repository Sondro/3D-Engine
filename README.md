# [3D Game Engine in Haxe ](https://github.com/Sondro/3D-Engine): 
[_Featuring Bullet Physics_](https://pybullet.org) 

### Controls:

|   KEY(S)   |          EVENTS         |
| :--------: | :---------------------: |
|    WASD    |          to move        |
|   ARROWS   |`           ^           `|
|    Q & E   |       rotate camera     |
|     F1     |        toggle FPS       |
|     O      |          Jump           |
|  SPACEBAR  |`	          ^           `|
|  NUMPAD9   |`           ^           `|
| Left Click |  enter fullscreen mode  |
|    Esc     |  exit fullscreen mode   |
